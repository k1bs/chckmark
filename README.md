## What is chk(mark) ?

chk(mark) is a note-taking app that makes your life a little bit richer. By using markdown, your notes come to life!

## Getting Started

Using chk(mark) is simple!

After creating your user, simply click the `+` icon to add your first note.

The provided text field allows you to type plain text or [Markdown](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet√Ç).

For instance, entering this:

```markdown
[I'm an inline-style link](https://www.google.com)
`inline.Code`
_this is italics_
```

renders as:

[I'm an inline-style link](https://www.google.com)  
`inline.Code`  
_this is italics_
